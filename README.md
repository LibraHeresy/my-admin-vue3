# my-admin-vue3

使用 Vue@3.5.13 + Antdv@4.x + Node@20.18.3 + Piana + Vite 技术栈

## 已实现功能

+ 登录页
+ 表单页
+ 异常页
+ 工作台
+ 国际化
+ 菜单
+ 路由
+ 404
+ 主题切换
+ 动态主题色
+ 图表引入
+ 富文本编辑器引入
+ 持久化
+ Mock
+ Lodash
+ TypeScript

## 安装环境

```
npm install yarn -g
yarn install
```

## 运行项目

```
yarn dev
```

## 打包项目

```
yarn build
```
